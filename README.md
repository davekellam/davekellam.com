Most of the code that powers the site

## Project layout

- `mu-plugins/`
  - `dk-core` houses functionality that persists across themes, including composer autoloader
  - `tsp-config.php` using theme switcher pro for theme migration
- `themes/spacemonkey/` old Underscores-based theme circa 2018.
- `themes/davekellam-25/` failed 25th anniversary theme 2025
- `themes/quarter/` new theme in progress 2026
- `plugins/dk-under-construction/` banner plugin
